# Microloans Service

A fully containerized microservice for managing micro-loans.  
Built using **Python (Flask/FastAPI)**, **PostgreSQL**, **Docker Compose**, and deployed with **GitHub Container Registry (GHCR)** through an automated **CI/CD pipeline**.

## üöÄ Features

- REST API for loan management  
- PostgreSQL as database  
- Automatic Alembic migrations  
- Environment-based configuration (`dev`, `staging`, `production`)  
- HTTPS-ready NGINX reverse proxy  
- CI/CD with GitHub Actions  
- Security scanning using Trivy  
- Auto-publishing Docker images to GHCR
